E-commerce Business Insights Dashboard

🔹 Overview

This project focuses on analyzing e-commerce sales data to uncover key business insights.
The goal is to help stakeholders make data-driven decisions by identifying trends, top-performing products, and revenue patterns.

🔹 Objectives

Analyze sales performance across categories
Identify top 5 products by revenue
Track high and low-performing categories
Provide actionable business insights

🔹 Tools & Technologies

SQL (Data extraction & querying)
Excel (Data cleaning & preprocessing)
Power BI (Dashboard & visualization)

🔹 Dataset

Source: kaggel
Data includes: Orders, Products, Sales, Revenue, Categories

🔹 Process

Data Cleaning (handled missing values, formatting)
Data Transformation (created calculated columns, KPIs)
SQL Queries (used joins, aggregations, filtering)
Dashboard Creation (interactive visuals in Power BI)

🔹 Key Insights

Top 5 products contribute to a major portion of total revenue
Certain categories consistently underperform
Seasonal trends observed in sales data
High-value customers drive repeat purchases

🔹 Business Recommendations

Focus marketing on top-performing products
Improve or remove low-performing categories
Offer discounts during low-sales periods
Target high-value customers with loyalty programs

Data Cleaning
Data Analysis
SQL Querying
Data Visualization
Business Insight Generation
